Conclusiones

Si NO forman parte de una expresión, entonces da igual usar ++cont que cont++

    for( var i=0; i<10; ++i ) { // instrucciones }
Funciona exactamente igual a: 
    for( var i=0; i<10; i++ ) { // instrucciones }

Si FORMAN parte de una expresión, entonces:

  alert( ++cont );
es equivalente a: 
  cont=cont+1;
  alert( cont );
Mientras que: 
  alert( ++cont );
es equivalente a: 
  alert( cont );
  cont=cont+1;
